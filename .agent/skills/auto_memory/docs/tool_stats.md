# Manual de Herramienta: The Dashboard (Stats)

## Propósito
Genera un informe visual del crecimiento de la memoria y el avance estratégico.

## Sintaxis (Powershell)
```powershell
powershell.exe -ExecutionPolicy Bypass -File ".agent/skills/auto_memory/scripts/stats.ps1"
```

## Salida Esperada
- Tabla de progreso estratégico (basada en `Auto_memory/INDEX_milestones.md`).
- Conteo de histories y sessions.
- Barra de progreso visual.
